# Java Shopping Cart with Localization

## Features
- Java console shopping cart
- Localization: English, Finnish, Swedish, Japanese
- UTF-8 support
- JUnit 5 tests
- JaCoCo coverage
- Docker support
- Jenkins CI/CD pipeline

## Run locally
```bash
mvn clean package
java -jar target/shopping-cart-localization-1.0-SNAPSHOT.jar
